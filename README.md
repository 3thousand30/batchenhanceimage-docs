# BatchEnhance Image — Documentation

Official documentation for [BatchEnhance Image](https://3thousand30.com/batchenhance-image.html), a batch image upscaling and print-preparation tool for Windows by [3thousand30](https://3thousand30.com).

## What is BatchEnhance Image?

BatchEnhance Image upscales and prepares entire image folders for high-quality print output. Set your target DPI, choose your output format, and process full batches in one run. Fully local — no cloud upload, no account, no telemetry. Nothing leaves your machine.

## Key features

- **Batch upscaling** — process full folders, not one image at a time
- **DPI-aware output** — set target DPI for print-ready results
- **Multiple output formats** — PNG, JPEG, TIFF, and more
- **Fully local** — no cloud, no account, no telemetry
- **No subscription** — buy once, use forever
- **Designed for print workflows** — Etsy sellers, poster makers, photographers, self-publishers
